### Introduction

```cpp
template<class T>
concept Integral = std::is_integral_v<T>;

void foo(Integral i) {
    std::cout << i;
}
```

* Invariants vérifiés par le type lui-même
* GCC : `-std=c++2a -fconcepts`

### Concepts vs std::enable_if
```cpp
// plutôt que :
template<class T>
using Integral = std::enable_if_t<std::is_integral_v<T>, T>;

template<class T>
void foo(Integral<T> i) {
    std::cout << i;
}
```
